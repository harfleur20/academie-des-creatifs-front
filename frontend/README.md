# Frontend Académie des Créatifs

## Objectif

Ce frontend constitue le socle React + TypeScript + Vite de la plateforme hybride :

- site public e-commerce ;
- espaces privés étudiants ;
- espace enseignant ;
- back-office admin.

## Commandes

```bash
npm install
npm run dev
```

Production :

```bash
npm run build
npm run preview
```

## État actuel

Le socle technique comprend :

- une base Vite propre ;
- un routage React structuré ;
- un shell visuel aligné avec le produit ;
- des pages de cadrage pour le catalogue, l'authentification et les dashboards.

## Prochaine étape

Brancher ce frontend sur l'API FastAPI pour remplacer les contenus statiques par de la donnée réelle.
